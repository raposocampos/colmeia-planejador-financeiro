# Jornadas do usuário

## Primeiro acesso

1. Conhecer as quatro ideias do produto.
2. Informar primeira conta e saldo, ou pular.
3. Optar conscientemente por dados fictícios.
4. Entrar no dashboard e receber próximos passos claros.

## Registrar uma despesa

1. Usar o atalho de despesa na visão geral.
2. Informar descrição, valor, data, categoria, conta ou cartão.
3. Salvar e receber confirmação.
4. Ver saldo, gasto por categoria e orçamento reagirem.

## Planejar uma categoria

1. Abrir Orçamentos e escolher o mês.
2. Definir categoria e limite.
3. Comparar utilizado, planejado e restante.
4. Receber mensagem educativa ao chegar a 80% ou ultrapassar.

## Construir uma meta

1. Informar objetivo, alvo, valor atual e data opcional.
2. Acompanhar percentual e restante.
3. Usar a estimativa mensal apenas como organização, nunca recomendação.

## Proteger os dados

1. Abrir Configurações.
2. Exportar backup JSON.
3. Ao importar, revisar quantidade e data do backup.
4. Confirmar explicitamente a substituição.
